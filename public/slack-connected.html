✅ Slack Connected

PostScout is ready

PostScout is now connected to your Slack workspace. It quietly builds context from conversations and suggests content ideas only when you ask.

How to get started

  • Add PostScout to a channel you want to track using /invite @PostScout
  • Chat as you normally would — no changes to your workflow
  • Run /suggest in that channel to get LinkedIn / X post ideas
  • Use /tokens to view token usage for your workspace

Privacy & data usage

PostScout only reads messages in channels it has been explicitly invited to. It does not auto-post, DM users, or read private conversations without access.

Conversation data is used only to build context and generate content suggestions. You stay in full control of what gets shared or published.
