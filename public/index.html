PostScout

PostScout is a Slack bot that turns real Slack conversations into LinkedIn and X post ideas. It listens quietly in channels you invite it to and only suggests content when you ask.

Connect PostScout to Slack

How it works

  • Invite PostScout to any channel using /invite @PostScout
  • Talk normally — product discussions, ideas, decisions, notes
  • Run /suggest to get 2–4 content ideas based on that conversation
  • Use /tokens to see token usage for your workspace

Open source

PostScout is currently a prototype and fully open source.

View GitHub repository →

Privacy & data usage

PostScout only reads messages in channels where it has been explicitly invited. It does not read other channels, DMs, or private conversations without access.

Messages are used only to build conversation context and generate content suggestions. PostScout never auto-posts or shares anything without you asking.

Built as a small experiment to explore turning conversations into content.
